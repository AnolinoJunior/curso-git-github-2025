git init .  -> Cria um repositório git na pastal atual
git status  -> Mostra o status do trabalho (de cada arquivo)
git add nome_arquivo -> Adiciona arquivo para ser commit
git commit - m "msg" -> Faz commit do que esta para ser commitado
git log  - > Mostra o histórico de commit
git diff - > Mostra todas mudanças adicionadas pro arquivo atual
git switch -> Troca a branch 
#Fluxo de trabalho (entender sobre comandos bash também)
1. criar o diretório (pasta) do projeto
2. iniciar o git dentro do diretório criado (git init.)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
